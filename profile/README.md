KUNLING (**Knowledge-augmented Universal Neural Learning for Intelligent Numerical Geoscience**) is an AI-driven numerical geoscience framework developed by Dr. Ze Liu’s research group at Ocean University of China.

聚焦知识增强的智能数值地学，核心研究方向包括：

- 多尺度数值建模与圈层耦合
- 智能数据同化与深时重建
- 构造—气候—沉积过程模拟
- 古地貌与古环境智能反演
- 全球多模态地学向量数据库

我们致力于融合物理模型、地学知识与人工智能，推动高效、可信、开放的地球系统研究。
